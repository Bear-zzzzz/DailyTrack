## 🎓 核心概念

### CoT（思维链）
**原理**：让 LLM 一步步思考，而不是直接给答案
```
❌ 直接：15% of 200 = 30
✅ CoT：让我们一步步思考... 15% of 200 = 0.15 × 200 = 30
```
**成效**：复杂推理任务准确率提升 20-40%

### ReAct 框架
```
Thought（思考） → Action（行动） → Observation（观察）
 ↑_________________________________________↓
```
**应用**：Agent 决策循环的核心

### 参数设置原则

| 参数 | 值 | 场景 |
|------|-----|------|
| temperature | 0.0-0.3 | 确定性任务（翻译、总结） |
| temperature | 0.7-0.9 | 创意任务（写作、头脑风暴） |
| top_p | 0.9 | 标准推荐 |
| top_k | 40-50 | 保留前 K 个概率最高的 token |
