## 💡 核心学习内容总结

### Week 1：LLM 基础理论
- Transformer 架构（Q/K/V、Self-Attention）
- 大模型训练流程（预训练→SFT→RLHF）
- Function Call 原理和 LoRA 微调

### Week 2：Agent 框架
- Chain vs Agent 的区别
- LangGraph 框架使用
- ReAct 框架和 3 个自定义工具实现

### Week 3：Prompt 工程
- CoT（思维链）和参数调优
- Prompt Caching 成本优化
- 不同推理范式对比

### Week 4：RAG 系统
- 完整的 RAG 链路设计
- 文档分割、向量化、混合检索
- Rerank 和幻觉规避

### Week 5：工作流设计
- 任务拆分策略
- **Text-to-SQL 系统**（今天焦点）
  - 3 个关键难点
  - 5 个最佳实践
  - Few-shot 示例构造
  - SQL 验证和纠错

### Week 6：多 Agent 系统
- 记忆管理（短期和长期）
- Agent 间协作和通信
- 反思改进循环

### Week 7：系统设计
- 5 个完整的系统设计案例
- 从业务需求到技术架构
- 系统设计面试讲解

### Week 8+：深化优化
- 项目性能优化
- 失败案例分析
- 面试最终准备
