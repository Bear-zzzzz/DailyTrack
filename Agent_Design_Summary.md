# 📊 Agent 设计总结（一页纸版本）

## 项目概览

```
项目名：LearningAssistant Agent（学习助手）
类型：智能问答系统
框架：LangChain + LangGraph
复杂度：初级-中级
预计时间：2-4 小时
```

---

## 核心工作流

```
用户提问："什么是 Attention？"
    ↓
[Agent 思考]
  "我需要搜索 Attention 的定义和应用"
    ↓
[Agent 行动]
  工具 1：搜索知识库 → 获得定义
  工具 2：获取相关概念 → Transformer, Self-Attention
  工具 3：获取代码示例 → Python 实现
    ↓
[Agent 判断]
  信息足够了吗？
  是 → 生成答案
  否 → 继续思考（循环）
    ↓
[生成答案]
  "Attention 是一种机制...
   相关概念有...
   代码实现如下..."
    ↓
用户获得完整答案
```

## 三个核心工具

```
┌─────────────────┬────────────────────┬──────────────────┐
│  SearchKB       │ GetRelatedConcepts │ GetCodeExample   │
├─────────────────┼────────────────────┼──────────────────┤
│ 输入: query     │ 输入: concept      │ 输入: concept    │
│ 输出: 相关文本  │ 输出: 概念列表     │ 输出: 代码片段   │
│                 │                    │                  │
│ 例：            │ 例：               │ 例：             │
│ 查询"Attention" │ Transformer        │ def attention(): │
│ → 定义、用途    │ → [Self-Attention, │    ...           │
│   等            │    Position Enc]   │                  │
└─────────────────┴────────────────────┴──────────────────┘
```

---

## 四个核心节点

```
┌──────────────────────────────────────────────────────────┐
│                   思考节点(Think)                        │
│  分析问题，决定需要哪个工具                              │
└──────────────────┬───────────────────────────────────────┘
                   ↓
┌──────────────────────────────────────────────────────────┐
│                   工具节点(Action)                       │
│  调用 SearchKB、GetConcepts、GetCode                    │
└──────────────────┬───────────────────────────────────────┘
                   ↓
┌──────────────────────────────────────────────────────────┐
│                   判断节点(Judge)                        │
│  是否获得足够信息？                                      │
│  ├─ 否 → 回到思考（循环）                              │
│  └─ 是 → 进入生成                                       │
└──────────────────┬───────────────────────────────────────┘
                   ↓
┌──────────────────────────────────────────────────────────┐
│                  生成节点(Generate)                      │
│  整合所有信息，生成最终答案                              │
└──────────────────────────────────────────────────────────┘
```
